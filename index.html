About

Welcome to the LPSWBD project (Launch Python Script with BATCH Demo) helps you launch python script with BATCH so you can turn python to exe faster (Turning BATCH to EXE can be done with bat2exe.net)! Edit the simple script with simple BATCH!

Simple Editing!

For non-BATCH experts all you have to change is the line: start c:\python%versionnum%\python run.py to start c:\python%versionnum%\python thenameofyourpythonprogram.py May change in newer versions!
